## 1. Home

방문자가 5초 안에 "뭐 하는 곳인지" 이해하고 바로 플레이하도록.

### Hero

- **헤드라인**: "내가 만든 코드가 즉시 게임 속에 반영되는 교육용 블록 코딩 플랫폼"
- **서브헤드**: Code Builder는 로블록스 게임 플레이 도중 블록 코딩 창을 열어 에이전트에게 명령을 내리는 인게임 코딩 교육 플랫폼입니다.
- **CTA**: [지금 플레이하기] → `https://www.roblox.com/join/1i9yw`
- **보조**: 데모 GIF (C키 → 블록 조립 → 에이전트 실행)

### 특징 4가지

| 특징 | 한 줄 설명 |
|------|-----------|
| PC · 모바일 지원 | C키(PC) 또는 `<>` 버튼(모바일)으로 코드 빌더 창을 열고 닫음 |
| 즉시 반영 | 블록을 조립하고 실행을 누르면 에이전트가 게임 속에서 즉시 반응 |
| 예측 불가 에이전트 | 에이전트가 항상 완벽하게 명령을 따르지 않음 — 그것도 게임의 재미이자 도전 |
| 12단계 커리큘럼 | 움직임 · 루프 · 조건문 · 함수를 다루는 12개의 점진적인 수업 |

### 작동 방식

**C키(또는 `<>` 버튼) → 블록 조립 → 실행** 순서의 스크린샷 시퀀스 3장.

### 하단 CTA

플레이하기 버튼 재노출. 문의가 있으면 Contact 링크.
